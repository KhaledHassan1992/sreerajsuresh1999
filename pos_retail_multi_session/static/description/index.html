MAIN FEATURE OF MODULE

  • In Original Odoo Point Of Sale: each User Account can opened only one POS Session of 1 POS Config, If need open second Session, required close last POS Session
  • It mean if you have many POS Counters, required create many POS Config, and Many Users Account, take a lot times for POS Setting
  • Main Feature of Module is: Allow You Create only one User, and Only one POS Setting and possible use at many POS Counter
  • Each POS Counter, User login via POS POS Pin on Each Employee Setting, 1 Employee is 1 POS Session of 1 POS Counter
  • So, Module can help User can open same session for same shop day by day separate with out close last day, if today the sales 14/8/2020 this session not closed ,Example: the User can open new Session for same shop for new day 15/8/2020 with out closing yesterday
  • Only 1 User Account Odoo can starting any POS Setting, you can create many POS Setting and assigned Employees can use
  • When POS Session started, POS Session just created has assigned by Employee Login, Employee can easy close last session or renew Session
  • Each POS config you can assigned many Employee, and Employee login session via POS Pin Pass (Config on Employee)
  • Manager can easy control, close all Session by one click time
  • Management can view Sale Summary each Employee (Employee create order save POS Order , POS Order Line for made Reporting)
  • With this module you Only Need 2 account user, 1 seller and 1 manager (admin) for take control all shop, all branch...
  • All Orders, Payment, Journal Entries ... save by Employee create Order
